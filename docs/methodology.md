# 방법론 노트

## 1. 프로젝트 목적

자산별 수익률 공분산과 포트폴리오 가중치로 리스크 기여도를 분해한다.

## 2. 핵심 질문

포트폴리오 변동성은 어떤 자산의 개별 변동성과 상관구조에서 주로 발생하는가?

## 3. 입력 데이터

- 파일: `data/sample/asset_return_panel.csv`
- 성격: 강의자료나 외부 원천을 복제하지 않은 합성 샘플
- 목적: 계산 흐름과 결과 해석을 작은 표본으로 먼저 검증

## 4. 계산 절차

1. 표본 공분산 행렬과 가중치 벡터로 포트폴리오 분산을 산출한다.
2. 각 자산의 marginal contribution과 component contribution을 구분한다.
3. 자산명과 수익률은 강의자료가 아닌 설명용 합성 패널이다.

## 5. 산출물 해석

`outputs/tables/baseline_results.csv`는 프로젝트별 baseline 결과입니다. 이 표는 최종 연구 결론이 아니라, 이후 실제 데이터 연결과 상세 분석을 시작하기 위한 구조 검증용 산출물입니다.

## 6. 한계

- 합성 데이터 기반이므로 통계적 유의성이나 투자 가능성을 주장하지 않습니다.
- 거래비용, 세금, 슬리피지, 유동성 제약은 일부 프로젝트에서만 단순 반영했습니다.
- 프로젝트별 정교한 모델링은 별도 후속 작업에서 확장해야 합니다.
